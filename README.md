<br/>"Переменные среды для ПК" - "Создать".
<br/>1 запись: Имя - "ANDROID_SDK_ROOT", значение - путь к папке с SDK (например D:\android\sdk )
<br/>2 запись: Имя "ANDROID_SDK_HOME", значение - путь к папке проектами (например D:\android\project )
<br/>![image](https://user-images.githubusercontent.com/52165649/142587588-32b09dcb-f968-4e94-8366-fadc24fcfcc1.png)
